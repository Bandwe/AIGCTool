# AIGCTool 使用说明

双击 **`启动.bat`** 打开窗口。顶部能看到 `ffmpeg / ffprobe / yt-dlp` 是否就绪。

---

## 一、拆分 / 合并

### 拆分
1. 把视频文件或文件夹**拖进**「拆分」子页（或用按钮选择）。拖文件夹会自动批量处理里面所有视频。
2. 选模式：
   - **按固定时长**：填秒数（如 `30`），从头到尾每 30 秒一段。
   - **按段数**：填段数（如 `5`），平均切成约 5 段。
3. 点「开始拆分」。结果在每个源视频旁的 `<视频名>_片段/` 文件夹里，命名 `001 002 …`。

### 合并
1. 把多个视频拖进「合并」子页（顺序 = 列表从上到下）。
2. 用「↑ 上移 / ↓ 下移」调顺序。
3. 点「开始合并」，选保存位置。

### 关于"无损"和切点
- 为保证片段能**逐帧还原**，切分只在**关键帧**处下刀。所以每段实际时长会**贴近**你填的秒数但不一定精确到秒（取决于关键帧间隔）。这是"不重编码 + 可还原"的必然取舍。
- 合并要求各视频编码/分辨率/参数一致（本工具自己拆出来的片段天然满足）。

## 二、多分辨率导出

1. 选「源视频」。
2. 勾选要导出的分辨率（4K / 1080 / 720，默认全选）。
3. 选编码：
   - **h265**（默认）：交付成片，体积小、观感无损。可调 CRF（越小越清晰、越大越省空间，默认 18）、preset、可勾「GPU 加速」用 NVIDIA NVENC。
   - **prores**：ProRes 422 HQ 母版，体积巨大，用于二次剪辑/调色。
4. 输出目录留空 = 源文件旁的 `output/`。
5. 点「开始导出」。统一锁 25fps CFR、16:9、Lanczos 缩放、BT.709。

## 三、视频下载

1. 在文本框里**每行粘一个链接**（B站 / YouTube 自动识别，可批量）。
2. **B站需要 cookies.txt**（否则可能返回 HTTP 412）：
   - Chrome/Edge 装扩展「Get cookies.txt LOCALLY」。
   - 登录 B站，在视频页点扩展图标 → Export，存成 `cookies.txt`。
   - 在「Cookie 文件」选它（留空会自动在 `D:\download`、下载夹等处查找）。
   - YouTube 一般不需要 Cookie，年龄限制/会员视频才需要。
3. 选保存目录（默认下载夹），点「开始下载」。装了 ffmpeg 会自动合并为 MP4。

下载失败常见原因：Cookie 过期（重新导出）、需要大会员（换会员账号 Cookie）、网络/代理问题。

---

## 四、屏幕录制

1. 「选择区域」框选录制范围，或「全屏」。
2. 选 FPS（15/24/30/60）和保存目录。
3. 点「● 开始录制」（窗口会自动最小化让出画面）；再点「■ 停止录制」结束。
4. **全局热键**：`Ctrl+F9` 开始/停止，`Ctrl+F10` 暂停/继续——录制时窗口最小化也能用。
5. 录完弹窗可一键打开所在文件夹。

> 录制输出为 mp4（mp4v 编码）。如需更高画质/更小体积，可把录好的视频拖到「多分辨率导出」再转一次。

## 常见问题

- **顶部显示 ffmpeg ✗**：拆分/合并/导出/下载合并都依赖它。`winget install ffmpeg` 装完重开窗口。
- **下载页是灰的**：没装 yt-dlp。`pip install -U yt-dlp`。
- **拖拽没反应**：`tkinterdnd2` 没装好，可改用按钮选择文件；或重跑 `启动.bat` 重装依赖。
- **启动闪退**：看本目录下的 `crash.log`。
